Quick Skill Summary

Provision a virtual card by posting merchant and amount. The API handles x402 negotiation and returns card details.

curl -X POST https://payper-card.onrender.com/api/run-agent \
  -H "Content-Type: application/json" \
  -d '{"merchant":"OpenAI","amount":20.00}'
