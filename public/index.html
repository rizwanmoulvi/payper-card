01

Agent Wallets

We leverage Circle Developer-Controlled Wallets. Your AI gets a secure, programmable MPC wallet. Private keys are never exposed in your execution layer.

02

x402 Payments

When the agent requests a card, the server responds with a 402 HTTP standard challenge. The agent automatically signs EVM messages and negotiates settlement natively.

03

Virtual Cards

Once the cryptic signature confirms on-chain execution via our custom Arc Facilitator, the system issues a single-use Visa card mapped exactly to the agent's invoice limit.

Drop-in HTTP endpoints.

The entire crypto complexity is abstracted. As long as your agent can trigger standard POST requests, it can negotiate x402 wrapped fetches dynamically to provision infrastructure.

  • POST /issue-card
  • GET /api/cards/:token
  • PATCH /api/cards/:token
// Provisioning a card programmatically
const fetchWithX402 = wrapFetchWithPayment(fetch, client);

const response = await fetchWithX402('/issue-card', {
  method: 'POST',
  body: JSON.stringify({
    merchant: 'AWS Web Services',
    amount: 50.00
  })
});

// Returns virtual PAN instantly
